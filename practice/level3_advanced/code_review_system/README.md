# 多 Agent 代码审查系统

基于多 Agent 协作架构的自动化代码审查系统。

## 🏗️ 架构设计

```
┌─────────────────────────────────────────────────────────────┐
│                    Review Controller                         │
│                   （审查控制器）                             │
└─────────────────────────────────────────────────────────────┘
                              │
        ┌─────────────────────┼─────────────────────┐
        ▼                     ▼                     ▼
┌───────────────┐    ┌───────────────┐    ┌───────────────┐
│ CodeReview    │    │  Security     │    │    Style      │
│   Agent       │    │    Agent      │    │    Agent      │
├───────────────┤    ├───────────────┤    ├───────────────┤
│ 逻辑正确性    │    │ 安全漏洞检测  │    │ PEP 8 规范    │
│ 算法效率      │    │ SQL 注入      │    │ 命名规范      │
│ 边界处理      │    │ 敏感数据泄露  │    │ 代码可读性    │
└───────────────┘    └───────────────┘    └───────────────┘
                              │
                              ▼
                    ┌───────────────┐
                    │  Summarizer   │
                    │    Agent      │
                    ├───────────────┤
                    │ 汇总报告      │
                    │ 优先级排序    │
                    │ 改进建议      │
                    └───────────────┘
```

## 🚀 快速开始

### 安装依赖

```bash
cd code_review_system
pip install openai
```

### 设置环境变量

```bash
export OPENAI_API_KEY=your_api_key
export OPENAI_BASE_URL=https://api.deepseek.com
```

### 使用方式

```bash
# 审查代码文件
python main.py -f your_code.py

# 直接传入代码
python main.py "def hello():\n    print('Hello')"

# 并行审查模式
python main.py -f your_code.py -p

# 显示所有 Agent 的详细结果
python main.py -f your_code.py -s
```

## 🔧 Agent 功能说明

### 1. CodeReviewAgent（代码逻辑审查）
- 逻辑正确性检查
- 算法效率评估
- 边界处理分析
- 潜在 Bug 检测

### 2. SecurityAgent（安全审查）
- SQL 注入风险检测
- 敏感数据泄露检查
- 命令注入风险检测
- XSS 漏洞检测

### 3. StyleAgent（代码风格审查）
- PEP 8 规范检查
- 命名规范检查
- 代码可读性评估
- 注释完整性检查

### 4. SummarizerAgent（报告汇总）
- 汇总所有审查结果
- 生成结构化报告
- 优先级排序问题
- 提供改进建议
