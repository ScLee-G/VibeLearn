# 多 Agent 任务分工系统

基于多 Agent 协作架构的智能任务分配和执行系统。

## 🏗️ 架构设计

```
┌─────────────────────────────────────────────────────────────┐
│                    Task Coordinator                         │
│                   （任务协调器）                             │
└─────────────────────────────────────────────────────────────┘
                              │
        ┌─────────────────────┼─────────────────────┐
        ▼                     ▼                     ▼
┌───────────────┐    ┌───────────────┐    ┌───────────────┐
│ TaskDecomposer│    │ TaskExecutor  │    │ TaskMonitor   │
│  （任务拆解）   │    │  （任务执行）   │    │  （任务监控）   │
├───────────────┤    ├───────────────┤    ├───────────────┤
│ 任务分析      │    │ 执行子任务    │    │ 进度跟踪      │
│ 依赖分析      │    │ 结果生成      │    │ 风险识别      │
│ 优先级排序    │    │ 错误处理      │    │ 瓶颈分析      │
└───────────────┘    └───────────────┘    └───────────────┘
```

## 🚀 快速开始

### 安装依赖

```bash
cd task_assignment_system
pip install openai
```

### 设置环境变量

```bash
export OPENAI_API_KEY=your_api_key
export OPENAI_BASE_URL=https://api.deepseek.com
```

### 使用方式

```bash
# 执行任务
python main.py "帮我创建一个博客网站"

# 并行执行模式
python main.py -p "帮我创建一个电商平台"

# 显示详细结果
python main.py -s "帮我分析市场数据"
```

## 🔧 Agent 功能说明

### 1. TaskDecomposer（任务拆解）
- 分析任务需求
- 拆解为子任务
- 分析依赖关系
- 优先级排序

### 2. TaskExecutor（任务执行）
- 执行具体子任务
- 生成执行步骤
- 错误处理和重试
- 返回执行结果

### 3. TaskMonitor（任务监控）
- 跟踪任务进度
- 识别风险和瓶颈
- 生成监控报告
- 提供优化建议
