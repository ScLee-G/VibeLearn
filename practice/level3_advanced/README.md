## 实践项目

### 项目 1: 知识库问答系统

**目标**：构建一个基于 RAG 的知识库问答系统

**技术要点**：
- 文档向量化
- 向量数据库查询
- 上下文注入

**文件**：
- `rag_qa_system.py`
- `knowledge_base/` - 知识库文档
