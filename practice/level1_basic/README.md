# Level 1: 入门探索

## 学习目标

- 掌握 LLM API 调用方法
- 理解 Prompt 工程基础
- 能够开发简单的 AI 应用

## 预计学习时长

2-3 周

## 难度评估

| 指标 | 评分 |
|:---|:---:|
| 概念理解 | ⭐⭐☆ |
| 代码复杂度 | ⭐☆☆ |
| 调试难度 | ⭐☆☆ |
| 创新性 | ⭐☆☆ |

## 实践项目

本阶段项目将帮助你建立 AI 应用开发的基础认知。建议按照以下路径探索：

### 推荐实践路径

1. 从 [LLM 基础概念](../../theory/llm_fundamentals/README.md) 开始学习
2. 掌握 [Prompt 工程](../../theory/theory_guide.md) 技巧
3. 尝试使用 OpenAI API 或其他 LLM API 构建简单应用

### 项目建议

- **简单聊天机器人**：学习 API 调用和对话管理
- **文本分类器**：练习 Prompt 设计和结果解析
- **代码补全工具**：探索代码生成能力
