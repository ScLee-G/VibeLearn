## 实践项目

### 项目 1: 智能代码生成器

**目标**：创建一个能够根据需求生成 Python 代码的 Agent

**技术要点**：
- 需求理解与代码生成
- 代码验证与调试
- 结果优化

**文件**：
- `code_generator.py`

---

### 项目 2: 数据处理 Agent

**目标**：实现一个自动化数据处理 Agent

**技术要点**：
- 地址转经纬度
- Excel 数据处理
- 距离计算

**文件**：
- `agent_workflow.py` - 核心工作流
- `sites_geocoded.xlsx` - 示例数据

---

### 项目 3: 自动化工作流

**目标**：构建一个多步骤自动化工作流

**技术要点**：
- 任务拆解
- 步骤编排
- 错误处理与重试

**文件**：
- `workflow_orchestrator.py`
