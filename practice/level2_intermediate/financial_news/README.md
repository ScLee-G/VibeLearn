# 金融新闻智能助手

> 每天早上自动获取并分析最新金融新闻，重点关注可能影响当天市场的内容。

## 📋 功能特性

- **定时推送**：每天早上 8:00 自动推送
- **智能筛选**：只推送可能影响市场的重要新闻
- **分类整理**：按重要程度分类（高/中/低）
- **影响分析**：每条新闻都有市场影响分析
- **多源聚合**：综合多个主流金融新闻源

## 🚀 快速开始

### 直接使用定时任务

这个项目已经配置好定时任务，每天 8:00 自动运行。

### 手动运行

```python
from financial_news import get_morning_brief

brief = get_morning_brief()
print(brief)
```

## 📊 新闻分类

| 重要程度 | 说明 |
|:---|:---|
| ⭐⭐⭐ | 重大政策、央行决策、重大数据发布 |
| ⭐⭐ | 重要公司新闻、行业政策 |
| ⭐ | 一般市场新闻、分析文章 |
