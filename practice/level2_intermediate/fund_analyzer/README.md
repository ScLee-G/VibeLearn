# 基金分析系统

VibeLearn Level 2 实践项目 - 每日市场基金涨跌分析与智能投资建议系统

## ✨ 新功能

- 📡 **真实数据源**：接入东方财富/天天基金网实时数据
- ⚙️ **自定义配置**：通过配置文件添加您自己的基金持仓
- 📊 **智能分析**：基于真实数据的市场分析和投资建议

## 功能特性

- 📈 实时市场概览分析
- 🏆 行业涨跌统计与TOP排行
- 💡 基于估值和技术分析的投资建议
- 🎯 精选基金推荐，包含仓位建议
- ⏰ 定时任务支持（每日14:40推送）

## 快速开始

### 1. 安装依赖

```bash
cd practice/level2_intermediate/fund_analyzer
pip install -r requirements.txt
```

### 2. 配置您的基金池

编辑 `fund_config.py` 文件，添加您关注的基金代码：

```python
MY_FUND_LIST = [
    "161725",  # 招商中证白酒
    "005827",  # 易方达蓝筹精选
    # ... 添加更多基金代码
]
```

### 3. 运行分析

```bash
python fund_analyzer.py
```

## 配置说明

### fund_config.py 配置项

- `MY_FUND_LIST`：您关注的基金代码列表
- `MY_HOLDINGS`：您的基金持仓信息（可选，用于个性化建议）
- `USE_REAL_DATA`：是否使用真实数据源（True/False）
- `DEFAULT_FUND_COUNT`：每次分析的基金数量
- `RECOMMENDATION_COUNT`：推荐基金数量

## 数据来源

- **东方财富/天天基金网**：实时基金估值和基本信息
  - API端点：fundgz.1234567.com.cn
  - 数据源：fund.eastmoney.com

## 使用提示

1. **交易时间**：建议在交易日14:40左右获取最新数据
2. **数据延迟**：实时估值数据仅供参考，最终净值以基金公司公布为准
3. **长期投资**：建议结合历史数据和市场环境综合判断
4. **风险提示**：投资有风险，请根据自身风险承受能力决策
