# 基金分析系统

VibeLearn Level 2 实践项目 - 每日市场基金涨跌分析与智能投资建议系统。

## 功能特性

- 📊 实时市场概览分析
- 📈 行业涨跌统计
- 🏆 涨幅/跌幅榜前列基金
- 💡 基于估值和技术分析的投资建议
- 🎯 5支精选基金推荐，包含仓位建议
- ⏰ 定时任务支持（每日14:40推送）

## 项目结构

```
fund_analyzer/
├── README.md              # 项目文档
├── fund_analyzer.py       # 主程序
├── requirements.txt       # 依赖配置
├── __init__.py           # 模块入口
└── examples/
    └── example_usage.py  # 使用示例
```

## 快速开始

### 1. 运行主程序

```bash
python fund_analyzer.py
```

### 2. 作为模块使用

```python
from fund_analyzer import get_daily_fund_report

report = get_daily_fund_report()
print(report)
```

## 自定义您的基金持仓

如果您希望基于自己持有的基金进行分析，可以修改 `fund_analyzer.py` 中的 `_init_mock_funds()` 方法，添加您实际持有的基金信息。

或者，您也可以：

1. 将您的基金持仓信息（基金代码、名称、当前持有仓位等）提供给我
2. 我可以帮您更新代码，生成更贴合您实际情况的投资建议

## 注意事项

⚠️ **重要提醒**：
- 当前使用模拟数据，实际使用时需要接入真实的基金数据API
- 投资建议仅供参考，不构成任何投资决策依据
- 投资有风险，请务必谨慎决策
- 建议结合自身风险承受能力和投资目标进行操作
