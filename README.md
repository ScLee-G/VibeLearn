# Vibe-Coding 学习仓库

> 让 AI 成为你的编程搭档，探索 AI 应用开发的无限可能

## 🎯 项目简介

本仓库是一个系统化的 Vibe-Coding（AI 辅助编程）学习框架，旨在帮助开发者掌握如何利用大语言模型（LLM）构建智能应用。

## 📚 学习框架

### 学习路径概览

| 阶段 | 名称 | 难度 | 预计时长 | 核心能力 |
|:---:|:---:|:---:|:---:|:---|
| **Level 1** | 入门探索 | ⭐⭐☆ | 2-3 周 | LLM API 调用、Prompt 工程基础 |
| **Level 2** | 进阶实践 | ⭐⭐⭐ | 3-4 周 | Agent 架构设计、代码生成 |
| **Level 3** | 高级应用 | ⭐⭐⭐⭐ | 4-6 周 | RAG 系统、复杂工作流编排 |

### 技术演进时间线（2023-2025）

| 阶段 | 时间 | 特点 | 代表技术 |
|:---:|:---:|:---|:---|
| **Phase 1** | 2023 | 过度迷信模型微调，崇尚行业垂直大模型 | 垂直领域微调 |
| **Phase 2** | 2024 | 明确了 RAG、Agent 的核心地位 | RAG、Agent、GPT |
| **Phase 3** | 2025 | Qwen、DeepSeek 胜任 AI 应用，技术路线形成成熟方法论 | 多模型兼容、商业化落地 |

## 🎓 学习路径

### Level 1: 入门探索

**目标**：掌握 LLM API 调用和基础 Prompt 工程

**学习内容**：
- LLM 基础概念与原理
- OpenAI API 使用方法
- Prompt 工程基础技巧
- 简单对话机器人开发

**实践项目**：
- 聊天机器人
- 文本分类器
- 代码补全工具

---

### Level 2: 进阶实践

**目标**：掌握 Agent 架构设计和 AI 代码生成

**学习内容**：
- Agent 核心组件（Prompt、LLM、工作流、代码执行）
- 任务拆解与规划
- AI 代码生成技术
- 结果验证与反馈

**实践项目**：
- 智能代码生成器
- 数据处理 Agent
- 自动化工作流

---

### Level 3: 高级应用

**目标**：掌握 RAG 和复杂系统架构

**学习内容**：
- RAG 检索增强生成
- 向量数据库应用
- 多 Agent 协作
- 企业级部署

**实践项目**：
- 知识库问答系统
- 多模态 Agent
- 企业级 AI 应用

## 🛠️ 技术栈

- **语言模型**：GPT-4、DeepSeek-V3、Qwen
- **框架**：LangChain、OpenAI SDK
- **向量数据库**：Pinecone、Chroma、Milvus
- **代码执行**：Python exec、Jupyter
- **数据处理**：Pandas、NumPy

## 📊 难度评估体系

| 指标 | 定义 |
