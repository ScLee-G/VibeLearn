## 🏆 能力等级定义

### Level 1：入门级
- 能够调用 LLM API 完成简单任务
- 理解基本 Prompt 设计原则
- 能够处理简单的文本生成任务

### Level 2：进阶级
- 能够设计 Agent 架构
- 能够实现 AI 代码生成和验证
- 能够构建完整的工作流

### Level 3：高级
- 能够构建 RAG 系统
- 能够实现多模态应用
- 能够部署企业级 AI 应用
