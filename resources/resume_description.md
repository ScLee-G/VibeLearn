# VibeLearn - 简历项目描述

## 📌 项目标题

**VibeLearn - 智能编程能力成长平台**

---

## 💼 简历项目描述（简洁版）

### 项目概述
> 构建了一个系统化的 AI 辅助编程学习平台，设计了从基础到高级的完整学习路径，帮助开发者掌握 LLM 应用开发能力。

### 技术栈
- **大语言模型**：GPT-4、DeepSeek-V3、Qwen
- **框架与工具**：LangChain、OpenAI SDK、Python
- **AI 技术**：RAG、Agent、Function Call、MCP
- **数据处理**：Pandas、NumPy

### 主要贡献
1. **学习框架设计**：设计了 Level 1-3 的渐进式学习路径，涵盖 LLM API、Agent 架构、RAG 系统等核心技术
2. **Harness 系统实现**：基于 Anthropic 三 Agent 架构，实现了包含初始化器、生成器、评估器的完整 AI 任务拆解与执行系统
3. **资源管理优化**：开发了内存/CPU 监控、性能追踪、缓存优化等组件，提升系统稳定性
4. **知识库构建**：整合了 RAG、Agent、Skill、MCP 等核心概念的完整文档体系

### 学习成果
- 掌握了从 Prompt 工程到多 Agent 协作的全栈 AI 应用开发能力
- 完成了 10+ 实践项目，包括智能代码生成器、数据处理 Agent、企业级 RAG 系统等
- 形成了系统化的 AI 应用开发方法论

## 🎯 技术面试官版（详细版）

### VibeLearn - 智能编程能力成长平台

**项目时间**：2025.05 - 至今

**项目描述**：
VibeLearn 是一个旨在帮助开发者系统性掌握大语言模型应用开发的学习平台。项目设计了从 LLM API 调用到企业级 AI 应用开发的完整学习路径，包含理论学习、实践项目、职业发展指南等模块。

**核心工作**：

1. **Harness 系统架构设计**
   - 基于 Anthropic 三 Agent 架构，设计了初始化器（Initializer）、生成器（Generator）、评估器（Evaluator）的协作模式
   - 实现了外部记忆管理、任务拆解、可验证检查点等核心机制
   - 开发了智能重试、错误恢复、渐进式上下文披露等容错机制

2. **RAG 与 Agent 系统实现**
   - 设计了检索增强生成的完整流程，包括向量存储、相似度检索、上下文注入
   - 实现了 Agent 工作流编排、状态管理、代码执行验证
   - 开发了 Model Context Protocol (MCP) 的标准化工具接口

3. **性能优化与资源管理**
   - 实现了内存、CPU、磁盘空间的实时监控与告警
   - 开发了响应缓存机制，减少重复 API 调用
   - 设计了性能追踪与分析系统，支持超时控制与资源配额

**技术亮点**：
- 采用声明式配置而非脚本驱动的部署模式
- 实现了 Feature List 和 Progress Log 的持久化状态管理
- 设计了可验证的检查点机制，确保 Agent 执行的可靠性

**学习成果**：
- 深入理解了 LLM、Agent、RAG、MCP 等核心 AI 技术
- 掌握了从单体到分布式的 AI 应用架构设计
- 形成了系统化的 AI 应用开发与调试方法论
